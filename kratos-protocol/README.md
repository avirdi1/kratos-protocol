# Kratos Protocol

A fitness and workout tracking app for strength training. Log your Push/Pull/Legs sessions, track sets/reps/weight, and monitor your progress over time.

## Prerequisites

- Node.js 18+
- npm

## Getting Started

```bash
# Clone the repo
git clone <your-repo-url>
cd kratos-protocol

# Install dependencies
npm install

# Start the dev server
npm run dev
```

The app will be available at `http://localhost:5173`.

## Build for Production

```bash
npm run build
```

The output will be in the `dist/` folder. You can preview it with:

```bash
npm run preview
```

## Tech Stack

- React 18 + TypeScript
- Tailwind CSS
- React Router v7
- Vite
